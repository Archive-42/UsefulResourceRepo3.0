-----------------------------------

<a href="/third-party-origin-trials/" class="w-card-base__link"></a>

Origin trials are a way to test a new or experimental web platform feature. A third-party origin trial makes it possible for providers of embedded content to try out a new feature across multiple sites.

<a href="/origin-trials/" class="w-card-base__link"></a>

Origin trials are a way to test a new or experimental web platform feature, and give feedback to the web standards community on the feature's usability, practicality, and effectiveness, before the feature is made available to all users.
