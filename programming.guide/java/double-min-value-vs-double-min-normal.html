Java: Double.MIN_VALUE vs Double.MIN_NORMAL

Double.MIN_VALUE represents the value 2−1074. This is a subnormal value, and it is the overall smallest possible value that a double can represent.

A subnormal value has a 0 before the binary point: 0.significand bits × 2exp

Double.MIN_NORMAL represents the value 2−1022. This is the smallest possible normal value that a double can represent.

A normal value has a 1 before the binary point: 1.significand bits × 2exp
